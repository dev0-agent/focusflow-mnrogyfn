# FocusFlow\n\n> A minimalist daily planner for focused professionals.\n\n## Overview\n\nFocusFlow is a client-side daily planner designed to help professionals organize their schedules and track progress with zero friction. Built with a minimalist philosophy, it removes the clutter of traditional project management tools and focuses purely on what needs to be done today. Users can quickly add tasks, assign priorities, and visually organize their day.\n\nThe application leverages a drag-and-drop interface to make rescheduling and prioritizing intuitive and fast. By running entirely in the browser with local storage persistence, FocusFlow ensures privacy, speed, and offline availability.\n\n## Tech Stack\n\n*   **Framework:** React + Vite\n*   **Styling:** Tailwind CSS\n*   **UI Components:** shadcn/ui\n*   **State Management:** Zustand (with local storage persistence)\n*   **Drag and Drop:** @hello-pangea/dnd (or @dnd-kit/core)\n*   **Icons:** Lucide React\n\n## Features\n\n*   ✨ **Minimalist Interface:** Distraction-free design focused on your tasks.\n*   🖱️ **Drag-and-Drop:** Easily move tasks between Todo, In Progress, and Done.\n*   🏷️ **Priority Labeling:** Visual color-coding for Low, Medium, and High priorities.\n*   📊 **Progress Tracking:** Visual progress bar showing daily completion percentage.\n*   💾 **Local Persistence:** Your data stays in your browser, instantly available.\n*   🌙 **Dark Mode:** Built-in support for comfortable low-light viewing.\n\n## Getting Started\n\n1.  **Install dependencies:**\n    ```bash\n    npm install\n    ```\n\n2.  **Start the development server:**\n    ```bash\n    npm run dev\n    ```\n\n3.  **Build for production:**\n    ```bash\n    npm run build\n    ```\n\n## Documentation\n\n*   [TASKLIST.md](./TASKLIST.md) - Project implementation tasks and progress\n*   [LEARNINGS.md](./LEARNINGS.md) - Developer notes and architectural decisions\n*   [.dev0/RULES.md](./.dev0/RULES.md) - AI agent rules and coding standards
